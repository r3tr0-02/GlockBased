<br>This mod adds Glock 19x into Altered Escape.

## INSTALLATION
Put `"Glock19x"` folder into `user/mods/Altered-Escape-2.2.0/items/weapon/`

Currently only works with Altered Escape's MelonLoader on JET 12.9
